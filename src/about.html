About this website

This website exists in two pieces: the website, which you're on right now, and the back-end that communicates with the Twitch.tv API to report the channel's online status.

It serves one simple purpose: extra accessibility to Cohh's Twitch.tv channel's online status, and a live viewer if it's online. An alternative name for this website is iscohhonline.com.

But why not just visit Twitch.tv directly?

  1. This website is available on GitHub, to serve as a learning tool for web platform basics. It includes SCSS, autoprefixer JS, npm scripts, CSS Grid and other modern tools that can get you started with web development.
